# tg_bot

Telegram-бот для выполнения различных задач и автоматизации процессов.

## Описание

Данный проект представляет собой базовую реализацию Telegram-бота, предназначенного для выполнения множества полезных действий. Вы можете расширить его функциональность добавлением новых команд и обработчиков событий.

## Установка

Для установки и запуска бота выполните следующие шаги:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/h4rd1/tg_bot.git
   ```

2. Перейдите в каталог проекта:
   ```bash
   cd tg_bot
   ```

3. Соберите Docker-образ:
   ```bash
   docker build -t h4rdl/telegram-bot -f ./Dockerfile .
   ```

4. Запустите контейнер:
   ```bash
   docker-compose up
   ```

## Конфигурация

Перед запуском убедитесь, что у вас есть токен Telegram-бота, полученный от BotFather.
